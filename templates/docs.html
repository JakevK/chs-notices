Usage:

Send requests to: https://chs-notices.herokuapp.com/notices

You will need an API key. How do you get an API key? I don't know ¯\_(ツ)_/¯

Put the API key on the end of the URL like this: https://chs-notices.herokuapp.com/notices?key=[your api key]


By default the notices will be taken from the most recent set of daily notices. To specify an older set of notices, use the date argument.

Use it something like this: https://chs-notices.herokuapp.com/notices?key=[your api key]&date=18-05-31

Make sure the date is in the format of yy-mm-dd.


And that's it. Good luck.
